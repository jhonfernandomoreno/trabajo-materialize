Comparativa entre Windows y Software libre

Software Libre Windows
Sin costes de licencia; los costes de asistencia dependen de las distribuciones Costes de licencia por usuario
Líneas de comandos Interfaz gráfica de usuario
Solución integrada (terminal y shell) Servidor de terminales; el cliente tiene que instalarse y configurarse
No ofrece portabilidad para todos los programas; gran cantidad de aplicaciones disponibles Soporta programas habituales; posibilidad de utilizar aplicaciones de Microsoft
Por lo general, pueden utilizarse los controladores de hardware para las distribuciones de Linux más tarde El nuevo hardware está diseñado normalmente para los sistemas Windows
Los usuarios habituales no tienen acceso a los ajustes básicos del sistema; las vulnerabilidades conocidas se solucionan rápidamente Elevado potencial de errores de usuario; interfaz integrada como posible punto de ataque
La asistencia varía en función de la distribución y de la versión Asistencia a largo plazo para todas las versiones
